# 📦 Sistema de Controle de Estoque

Sistema completo para gerenciamento de estoque, desenvolvido em Python com Streamlit. Ideal para pequenos negócios que desejam controlar seus produtos, movimentações e gerar relatórios profissionais.

## ✨ Funcionalidades

- ✅ Cadastro de produtos com código único
- ✅ Controle de entrada e saída
- ✅ Histórico de movimentações
- ✅ Alertas de estoque baixo
- ✅ Dashboard com gráficos
- ✅ Relatórios em PDF personalizados
- ✅ Interface web responsiva
